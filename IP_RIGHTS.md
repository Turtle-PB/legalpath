**What qualifies as a trade secret:**
- ✅ The analysis engine algorithms
- ✅ The clause detection patterns (400+ keywords)
- ✅ The document template generation logic
- ✅ The chatbot response system

**What you must do to maintain trade secret protection:**
1. ✅ Mark documents as "CONFIDENTIAL — Trade Secret"
2. ✅ Use access controls (password gate on the app)
3. ✅ Keep source code private when possible (consider obfuscation for deployed version)
4. ✅ Use NDAs when sharing with collaborators
5. ✅ Add copyright headers to all files

**If someone steals your trade secret:**

**⚠️ WARNING:** Publishing source code publicly (on GitHub) may weaken trade secret claims for the published portions. The analysis engine logic is visible in the JavaScript. Consider:
- Keeping a "pro" version with enhanced algorithms private

### 5. 🌐 DMCA — TAKEDOWN RIGHTS

**What it means:** If someone copies LegalPath and puts it on a website, you can force the hosting provider to remove it within 48 hours — for free.

**The law says:**
- **17 U.S.C. § 512(c)(3)** — DMCA takedown notice process

**How to file a DMCA takedown:**
1. Find the infringing URL
2. Identify the hosting provider (WHOIS lookup)
3. Send a DMCA notice to their copyright agent
4. They must remove it within 48 hours
5. If they don't, you can sue
